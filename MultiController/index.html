MultiController for Mac

Created By: Salty | Zed | Zalty | Captain Salty Electrozapper

Home of the first Mac MultiController for ToonTown

Features:

  • Mirror Mode for 4 toons
  • Multi Mode for 2 groups of 2 toons
  • Keep Alive
  • Custom Keymaps

Future:

  • Other stuff people want

How to use

  • Click the top left target, then select the toontown window on the left. The target will turn green if successful
  • Repeat for the top right target, as well as the other two targets underneath for group 2
  • In the App Menu, click Controller > 'Toggle Controller' (or use the shortcut ⌘` to enable
  • Controls:
    • ⌘` -> Enable/Disable listeners
    • ` -> switch between modes
      • Mirror Mode
        • \ -> Throw pies on both toons
        • Broadcasts every other key press to both processes
      • Multi Mode
        • ESC -> Opens book on both toons
        • 1 / 2 -> Select group 1 or 2
        • \ -> Throw pies on both toons (can be configured individually)
        • Left Toon:
          • Left Shift -> Jump
          • WASD -> Move toon
        • Right Toon:
          • Right Shift -> Jump
          • Arrows -> Move toon
