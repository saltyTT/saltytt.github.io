How to use

  • Click the top 'Left' button, then select the toontown window on the left. The button will turn green if successful
  • Repeat for the 'Right' button, as well as the other Left and Right buttons underneath for more toons
  • In the App Menu, click Controller > 'Toggle Controller' (or use the shortcut ⌘` to enable
  • Controls:
    • ⌘` -> Enable/Disable listeners
    • ` -> switch between modes
      • Mirror Mode
        • Broadcasts every key press to both processes
      • Multi Mode
        • ESC -> Opens book on both toons
        • Left Toon:
          • Left Shift -> Jump
          • WASD -> Move toon
        • Right Toon:
          • Right Shift -> Jump
          • Arrows -> Move toon
