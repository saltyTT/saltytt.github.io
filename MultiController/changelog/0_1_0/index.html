0.1.0

March 24 2018
  • Added Multi Mode

Usage:

Press ` (grave/tilde key above tab) to switch between modes
  • Mirror Mode
    • Broadcasts every key press to both processes
  • Multi Mode
    • escape opens book on both toons
    • Left Toon:
      • Jump: Left Shift
      • Move: WASD
    • Right Toon:
      • Jump: Right Shift
      • Move: Arrows
