Set Up

  • Click the top left target, then select the toontown window on the left. The target will turn green if successful
  • Repeat for the top right target, as well as the other two targets underneath for group 2
  • In the App Menu, click MultiController > 'Preferences' (or use the shortcut ⌘,) to configure controller
  • In the App Menu, click Controller > 'Toggle Controller' (or use the shortcut ⌘`) to enable the controller

Controls:

Anything marked with * is configurable

  • Global
    • ⌘Q : Quit MultiController
    • ⌘R : Relaunch MultiController
    • ⌘, : Open Preferences
    • ⌘` : Toggle MultiController
    • ` : Switch Modes
  • Mirror Mode
    • \ : Throw pies *
    • Broadcasts every other key press to both processes
  • Multi Mode
    • 1 : Select group 1
    • 2 : Select group 2
    • Escape : Open book on both toons
    • Left Toon
      • Left Shift : Jump *
      • WASD : Move toon *
      • \ : Throw pie *
    • Right Toon
      • Right Shift : Jump *
      • Arrows : Move toon *
      • \ : Throw pie *
